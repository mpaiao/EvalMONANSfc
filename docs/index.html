MONAN Evaluator (Surface)

Marcos Longo

25-Feb-2026

These documents and scripts can be used to help assess MONAN simulations that have been post-processed with scripts_CD-CT against station data. The processing has a mix of shell scripts and R scripts.

  • 01_gen_ncdf_intercomp.bash. This shell script extracts a sub-domain of MONAN output files to a region of interest. This is useful when running wall-to-wall comparisons for a specific sub-domain.
  • 02a_ExtractINMET.Rmd. This R Markdown takes automatic weather station data from Brazil’s National Institute of Meteorology (INMET), organise them into more standard csv files and create a look-up table.
  • 02b_ExtractMETAR.Rmd. This R Markdown takes airport weather (METAR) data obtained upon request from MASTER, organise them into csv files and create a look-up table.
  • 03_ExtractMETAR.Rmd. This shell scripts extract MONAN forecasts for each site with data available during the assessment period.
  • 04_EvalMONAN_SfcStation.Rmd. This R Markdown produces a series of comparisons between the model and the observations, and generates multiple plots to help assess the model.

Important notes

  1. Make sure to download the directory “RUtils” available on GitHub, and set variable util_path accordingly for R Markdown scripts that have this variable.
  2. In the future, we will try to simplify the workflow to reduce the number of steps. We will also add alternative steps 2 that can handle files pooled and curated by Brazil’s National Institute for Space Research (INPE).
  3. These scripts require multiple R packages, Climate Data Operators (CDO), netCDF Operators (NCO) and NCAR Command Language (NCL).
